# 一.从零搭建一个 Vue 项目

# 1.首先安装 Vue-cli3.0

在安装 Vue-cli3.0 之前需要安装 node 和 npm

node 下载

WARNING

Vue-cli3.0 需要在 Node.js8.9 或更高版本

  • 安装完毕，打开 cmd，通过查看 node 版本来判断是否安装成功
node -v
1
  • 在安装 node 的时候，npm 包管理器会自动安装，通过查看 npm 版本来判断是否安装成功
npm -v
1
  • 全局安装 Vue-cli3.0
npm i -g @vue/cli
# 或者使用yarn安装 yarn install -g @vue/cli
vue --version # 查询版本是否为3.x

# 3.运行项目

下载依赖

npm i
# 或者 yarn install
1
2

运行项目

npm run dev
1

项目目录结构

├── node_modules     # 项目依赖包目录
├── public
│   ├── favicon.ico  # ico图标
│   └── index.html   # 首页模板
├── src
│   ├── assets       # 样式图片目录
│   ├── components   # 组件目录
│   ├── views        # 页面目录
│   ├── App.vue      # 父组件
│   ├── main.js      # 入口文件
│   ├── router.js    # 路由配置文件
│   └── store.js     # vuex状态管理文件
├── .gitignore       # git忽略文件
├── .postcssrc.js    # postcss配置文件
├── babel.config.js  # babel配置文件
├── package.json     # 包管理文件
└── yarn.lock        # yarn依赖信息文件
